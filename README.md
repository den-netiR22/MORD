# Dark Fun **Добро пожаловать на сервер Dark Fun.**

## Правила сервера **Обязательно к прочтению**

1. В дуэльных зонах строго запрещён FFA и на всей карте вне дуэльных зон действует режим FFA.
2. Использование команд VIP, MOD, ADMIN и SUPER ADMIN с целью получения игрового преимущества карается снятием привилегий.
3. Teaming, буллинг, хантинг и неподобающее поведение запрещены, любой игрок может оставить отзыв, жалобу или предложение в нашем Discord.
